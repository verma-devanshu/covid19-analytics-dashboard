# COVID-19 Analytics Dashboard

A Streamlit-based dashboard for analyzing COVID-19 datasets through visualization, trend analysis, and machine learning techniques.

## Live Demo

🚀 https://covid19-analytics-dashboard.streamlit.app/

## Features

- Country-wise case analysis
- Recovery and mortality trends
- Demographic analysis
- Interactive visualizations
- PCA-based dimensionality reduction
- K-Means clustering

## Tech Stack

- Python
- Streamlit
- Pandas
- NumPy
- Scikit-learn
- Plotly
